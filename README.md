# Frontend — Vibe Route

Interfaz web desarrollada con **React 18** y **TypeScript**, empaquetada con **Vite**.
Incluye visualización de mapas, gestión de pedidos y panel de rutas optimizadas.

## Comandos

> Aquí van los comandos para levantar el servicio; rellenar tras inicializar el proyecto con Vite.

```bash
# Ejemplo:
# npm install
# npm run dev
```
